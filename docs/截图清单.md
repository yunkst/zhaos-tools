## 如何截图

### Windows 版本

1. **运行应用**：
   ```bash
   cd D:/myspace/teacher_tools/app
   flutter run -d windows
   ```

2. **截图方法**：
   - 使用 Windows 截图工具：`Win + Shift + S`
   - 或使用第三方工具（如 Snipaste）

### Android 模拟器

1. **运行应用**：
   ```bash
   cd D:/myspace/teacher_tools/app
   flutter run -d emulator-5554
   ```

2. **截图方法**：
   - Android Studio 自带截图工具
   - 或使用 adb 命令：`adb shell screencap -p /sdcard/screen.png`

### 保存位置

将截图保存到：
```
D:/myspace/teacher_tools/docs/images/
```
