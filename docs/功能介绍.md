## 1. 首页仪表板

首页是应用的中心枢纽，提供快速访问各项功能的入口。

### 功能特点

- **班级切换**：顶部下拉菜单快速切换当前管理的班级
- **统计卡片**：显示学生总数、班级平均分等关键数据
- **快速操作**：一键访问常用功能（记录随笔、成绩录入等）
- **最近记录**：展示最近的随笔记录，方便快速查看

**操作提示**：
- 点击顶部班级名称可切换到其他班级

## 2. 学生档案管理

完整的学生信息管理系统，支持灵活的搜索和筛选。

### 2.1 学生列表

### 功能特点

- **智能搜索**：支持多种搜索方式
  - 姓名：如 "张三"
  - 拼音：如 "zhangsan"
  - 首字母：如 "zs"
  - 学号：如 "2024001"
  - 混合搜索：如 "zhang 2024"

- **高级筛选**：
  - 按性别筛选：男生 / 女生
  - 按职务筛选：班干部 / 普通学生

- **排序选项**：
  - 按姓名排序
  - 按学号排序
  - 按最近记录时间排序

- **快捷操作**：
  - 长按学生卡片：弹出快捷菜单（记录随笔、查看成绩、编辑、删除）
  - 点击学生卡片：查看学生详情

### 2.2 学生详情

展示学生的完整档案信息。

### 信息分类

| 分类 | 内容 |
|-----|------|
| **基本信息** | 姓名、学号、性别、年龄 |
| **身体特征** | 身高、视力 |
| **家长信息** | 两位监护人的姓名、电话、称谓、工作单位、职务 |
| **学校职务** | 班干部、家委会成员等 |
| **随笔历史** | 按时间倒序展示所有相关记录 |

### 界面预览

**操作提示**：
- 点击"编辑"按钮修改学生信息
- 点击"记录随笔"快速创建新随笔
- 点击"查看成绩"浏览该学生所有考试成绩

---

### 2.3 学生表单

新增或编辑学生信息。

### 必填字段

- 学号（班级内唯一）
- 姓名
- 家长1姓名
- 家长1电话

### 可选字段

性别、出生日期、身高、视力、家长信息（第二监护人）、职务、获奖情况等，共28个字段。

## 3. 随笔记录管理

灵活的记录系统，帮助教师追踪学生的日常表现。

### 3.1 随笔列表

### 功能特点

- **类型筛选**：
  - 表现：课堂表现、行为举止
  - 作业：作业完成情况
  - 考勤：出勤记录
  - 活动：活动参与情况
  - 其他：自定义类型

- **标签系统**：
  - 积极：正向评价
  - 表扬：值得表扬的行为
  - 课堂：课堂相关
  - 违纪：需要改进的行为
  - 支持自定义标签

### 界面预览

>

### 3.2 创建随笔

快速记录学生事件。

### 操作步骤

1. 选择学生（支持搜索）
2. 选择类型（表现/作业/考勤/活动/其他）
3. 选择或添加标签
4. 填写内容
5. 保存

### 界面预览

> 📸 **截图位置**：`docs/images/05-note-create.png`
>
> <!-- 截图说明：展示创建随笔页面，包含学生选择器、类型选择、标签选择、内容输入框 -->

**快捷入口**：
- 首页 → 快速操作 → 记录随笔
- 学生列表 → 长按学生卡片 → 记录随笔
- 学生详情 → 记录随笔按钮

## 截图指南

### 如何添加截图

1. **运行应用**：
   ```bash
   cd D:/myspace/teacher_tools/app
   flutter run -d windows
   # 或
   flutter run -d emulator-5554
   ```

2. **截图要求**：
   - 分辨率：1920x1080
   - 主题：浅色
   - 格式：PNG
   - 包含测试数据

3. **保存位置**：
   - 将截图保存到 `docs/images/` 目录
   - 按照上述表格中的文件命名

4. **更新文档**：
   - 截图添加后，将 `<!-- 截图说明：... -->` 替换为：
     ```markdown
     ![功能说明](images/xx-filename.png)
     ```
